Udacity Project

Unscramble Computer Science Problems

A Python analysis project that solves five phone-record tasks using the provided September 2016 call and text dataset, with a written runtime analysis for each solution.

The repository is designed as a clean, review-ready course submission and portfolio sample.

What It Covers

  • Record inspection and dataset summarization
  • Distinct-number counting
  • Total call-duration analysis
  • Bangalore fixed-line code extraction
  • Telemarketer candidate detection
